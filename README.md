This repository is used to host release artifacts & dSYMs for [VietnameseInput][Repository]. Each tag matches an SDK version.

The [VietnameseInput website][Website] has information about the SDK, a blog with news and updates, pricing info, etc.


## Project

The [VietnameseInput repository][Repository] has a Swift Package that can be used to add Vietnamese typing support to apps.
